Using External Subtitle Support

You can use .ssa substation alpha, .srt SubRip, or .sub MicroDVD subtitling files in niceplayer. To use them, the files must share the same name as your video content except for the file extension. Support is still preliminary, there aren't many options and not all subtitling features are supported. The subtitles once recongnized by NicePlayer maybe turned on and off from the 'Movie' menu or by a right click.
